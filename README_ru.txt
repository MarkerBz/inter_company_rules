НАСТРОЙКА МУЛЬТИКОМПАНИЙ =====================================:

- для v8 - установить модуль 'multi_company'; в v9/10 встроен!
- в Settings–>General Settings:
	- V="Manage multiple companies"
- Настройки -> Компании -> Структура компании
	- иерархию создать! (одна комп всегда Parent!?)
- in Settings–>Users:
	- создать пользователей разных компаний
	- V="Multi Companies"
	    - если не указать, ~будет "видеть" только своё (иначе +родителя)
	- указать компанию
	- указать разрешённые компании
- склады / (?)места хранения(?) создавать под определённым пользователем!:
	(? или можно с настройками склада - "Один склад и одно место хран?")
	- in Склады -> Настройка -> Склады (или) Места хранения
	- указать компанию склада (или автоматом?)
	http://www.odoohelp.com/multi-company-configuration-in-openerp-7/

НАСТРОЙКИ ДЛЯ МОДУЛЯ 'inter_company_rules' ===================:

- создать пользователя ДРУГОЙ компании (~admin2) - от его имени будут
	создаваться (автоматически) док-ты другой компании.
- нужным Пользователям - добавить права "Склад" == Менеджер! +F5 в браузере!
- в нужных Контрагентах ('res.partner') заполнить поле 'Supplier Currency'
	(если этот партнёр будет "автоматическим" поставщиком)
- для функционала создание "Invoice/Refunds":
	- для компаний на/от которых будут Инвойсы:
	1. создать бухгалтерские журналы в меню:
		Бух.учёт -> Настройки -> Бух.учёт -> Журналы
		(лучше создать КОПИРОВАНИЕМ существующих двух записей)
			Счета заказчику 	<Компания>
			Счета от поставщиков 	<Компания>
	2. настроить бухгалтерский план счетов в меню:
		Бух.учёт -> Настройки -> Настройки -> План счетов
			- в поле "План счетов" должно быть что-то типа:
				"Configurable Account Chart Template"
	3. в товарах ('product.template') которые будут в Инвойсах:
		- проставить "Налоги с покупателя" и "Налоги поставщика"
- Для отладки (для УДАЛЕНИЯ инвойсов) можно установить мод 'account_cancel',
	и в бух. журналах проставить галочку "Allow Cancelling Entries".

- в Компаниях на/от которых будут документы:
	- валюта компаний ~одинаковая!
	- на закладке "Inter-Company Rules":
		- настроить флаги
		- заполнить поле "Inter Company User"
- (для v9+) Настройки -> ~'Share products ...'=V
	- для v8: в Товарах - НЕ указывать компанию!!!
- настроить БУХ - счета и журналы!!! (иначе док-ты НЕ подтверждаются):
	- для верхней галочки - "журнал покупок" по фирме
	- для нижней галочки - счета внутри товара

- (опционально) "Продвинутые" доп. настройки по каждой компании:
	Настройки -> Параметры -> Параметры компании
